Defina a pasta base que contém o repositório de documentos. Use o botão para abrir o seletor de pastas (File System Access API) quando disponível, ou selecione uma pasta via o seletor de ficheiros com webkitdirectory como fallback (navegadores Chromium).

Ações Perigosas

Operações irreversíveis sobre os dados da aplicação.

Apagar todo o conteúdo
Remove permanentemente todas as pessoas, famílias, fontes, repositórios, notas e multimédia registados na base de dados. O histórico de ações também será limpo. Esta operação não pode ser desfeita.
Limpar todas as definições
Repõe todas as definições da aplicação (pessoa em foco, pastas configuradas, preferências) para os valores padrão. Os dados genealógicos não são afetados.
